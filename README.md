## Description :

Ce dépôt contient le "**Projet Flash**" réalisé dans le cadre de la formation à la [**Coding Factory**](https://codingfactory.fr/) by [**ESIEE-IT**](https://www.esiee-it.fr/fr). L'objectif est de concevoir un site de mini-jeu, avec les technologies de base du développement web : **HTML, CSS, JavaScript, PHP, MySQL**.


## Étape actuelle du développement :

Le **projet** est actuellement à sa **phase n°2** de conception, la partie __base de données / SQL__. 

## Installation : 

Pour débuter l'installation, 

- Accéder au dépôt **GitHub** : https://github.com/maxenceb91/Projet-Flash

- Cloner le projet :
  - Ouvrir un **nouveau terminal**, vérifier qu'il soit sur le **bureau**. Accédez
  - Faire la **commande** : ```git clone https://github.com/maxenceb91/Projet-Flash```
  - Le projet doit apparaître dans un dossier appelé "**Projet-Flash**"
  - Vous pouvez ensuite accéder au site en double cliquant sur "index.html" ou "index.php" à la racine du dossier "**Projet-Flash**".

## Détails :

Le site a été construit selon la **maquette** fournie par notre enseignant.
La base de données a été construite pour répondre aux besoins des utilisateurs.

Fonctionnalité back end couverte via les requêtes SQL : 

-  Initialisation de la BDD
- Gestion des utilisateurs (Authentification)
- Gestion du jeu et des scores
- Chat général & privé
- Messagerie privée
- Statistiques
